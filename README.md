# Atividade: Blueprint do Downstream

**Disciplina:** Análise, Projeto e Desenvolvimento Ágil

**Aluno:** Vytor de Oliveira

## Descrição
Este repositório contém a documentação estratégica e os arquivos de configuração para o processo de Downstream (do código à produção), focando em qualidade, automação e previsibilidade.

## Estrutura dos Arquivos
- **Blueprint.md:** Visão geral da estratégia de engenharia, testes e observabilidade.
- **.github/workflows/ci.yml:** Definição da automação do pipeline (CI/CD).
- **PULL_REQUEST_TEMPLATE.md:** Checklist de qualidade (DoD/DoR) para revisão de código.
- **POLITICA_Promocao.md:** Regras para mover código de Dev -> Staging -> Prod.
- **RUNBOOK_rollback.md:** Guia de procedimentos para reversão em caso de falha.
